# GranaSAT Dashboard

GranaSAT Dashboard project aims to control our Ground Station via web, including:

- Antenna Rotors
- Kenwood TS-2000 radio transceiver
- ICOM 9100 radio transceiver
- TNC
- Sotfware Defined Radios
